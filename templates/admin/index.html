HemelB Offload Service

This service provides a set of simple REST endpoints to allow remote execution and management of HemelB simulations on EPCC's CIRRUS service and SURFSara's LISA service

The service is used by the PolNet workflow suite to allow computationally demanding simulations to be executed remotely rather than on the user's local workstation

This work is funded under CompBioMed Work Package 5.
